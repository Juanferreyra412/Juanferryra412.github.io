Sobre el TPO:
Las páginas diseñadas para el TPO consisten en la página de una empresa ficticia que se dedica 
a la prestación de servicios relacionados a la electrónica.
Consignas:
En la página principal hay una transición que usa transform: scale(). Esta se encuentra aplicada
a 3 div que están sobre el formulario de contacto. Esta transición se activa colocar el cursor
sobre los elementos.
El la página "Otros proyectos" hay una animación realizada con @keyframe que hace que se muevan 
tres círculos de colores.
